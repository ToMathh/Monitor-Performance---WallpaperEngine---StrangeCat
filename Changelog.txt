INSTALLATION RAPIDE
───────────────────
  1. pip install psutil pillow pystray pywin32 wmi
  2. python make_icon.py          (génère icon.ico)
  3. python strangecat_monitor.py (lancer l'app)

  — OU pour créer un EXE standalone :
  4. double-cliquer build.bat
  5. récupérer dist\StrangeCat Monitor.exe

CPU TEMP — ORDRE DE PRIORITÉ
─────────────────────────────
  1. psutil            (sans app tierce, Linux/certains OEM)
  2. WMI/AcpiThermal   (sans app tierce, ACPI BIOS)
  3. WMI/PerfData      (sans app tierce, compteurs Windows)
  4. LibreHardwareMonitor / OpenHardwareMonitor  (app tierce)
  5. CoreTemp           (app tierce, mémoire partagée)

  → Pour avoir la temp CPU sous Windows, installer
    LibreHardwareMonitor et le lancer en Administrateur.

HTTP API
────────
  GET http://127.0.0.1:5100/performance  →  JSON complet
  Champ : "cpu_temp" (null si non disponible),
           "cpu_temp_method" (source utilisée)
